¿Qué te parece el curso?

BRONCE 3: Hola Mundo

1. ¿Qué es Python?

Un lenguaje fácil de leer, poderoso y muy popular. Ideal para empezar.

2. Tu primer comando

Usa print("Hola Mundo") para mostrar texto en la pantalla.

BRONCE 2: Variables

1. ¿Qué es una variable?

Es como una "caja" en la memoria donde guardas información. Ej: edad = 20

2. Reglas para nombrarlas

No pueden empezar con números ni tener espacios (usa guion_bajo).

BRONCE 1: Tipos de Datos

1. Strings (Cadenas)

Cualquier texto entre comillas. Ej: nombre = "Goku"

2. Enteros y Flotantes

Números sin decimales (int) y con decimales (float). Ej: pi = 3.1416

PLATA 3: Operadores

1. Matemáticos

Suma +, Resta -, Multiplicación *, División /.

2. Lógicos

Usa and, or y not para combinar condiciones.

PLATA 2: Condicionales I

1. El bloque if

Si una condición es verdadera, ejecuta el código. if edad >= 18:

2. El bloque else

Si la condición del "if" es falsa, entra aquí por defecto.

PLATA 1: Condicionales II

1. El bloque elif

Te permite evaluar múltiples condiciones en cadena antes de llegar al else.

2. Anidación

Colocar un if dentro de otro if. Recuerda cuidar la indentación.

ORO 3: Ciclo For

1. Iteración

Recorre elementos uno por uno. for letra in "Hola":

2. Función range()

Crea secuencias de números. for i in range(5): repetirá 5 veces.

ORO 2: Ciclo While

1. Bucle por condición

Ejecuta código mientras la condición sea True. while vidas > 0:

2. Bucles infinitos

Cuidado con no cambiar la variable de control, o el programa se trabará.

ORO 1: Listas

1. Colecciones

Guarda varios datos en corchetes. nombres = ["Ana", "Juan"]

2. Métodos útiles

Agrega elementos con .append() o elimínalos con .remove().

PLATINO 3: Tuplas y Sets

1. Tuplas

Como las listas, pero no se pueden modificar. Usan paréntesis ().

2. Sets (Conjuntos)

Colecciones sin orden que automáticamente eliminan datos duplicados.

PLATINO 2: Diccionarios

1. Clave y Valor

Guardan datos con etiquetas. auto = {"marca": "Ford", "año": 2020}

2. Acceso a datos

Para ver el valor, llamas la clave: print(auto["marca"]).

PLATINO 1: Funciones I

1. Definición

Empaqueta código para reutilizarlo usando la palabra def.

2. Parámetros y Return

Recibe datos en los paréntesis y devuelve resultados con return.

DIAMANTE 3: Funciones II

1. Scope (Alcance)

Diferencia entre variables globales (todo el código) y locales (solo en función).

2. Funciones Lambda

Funciones anónimas de una sola línea. Rápidas y elegantes.

DIAMANTE 2: Manejo de Errores

1. Try / Except

Atrapa errores ("crashes") sin que el programa se cierre de golpe.

2. Finally

Un bloque de código que se ejecuta siempre, haya ocurrido error o no.

DIAMANTE 1: Archivos y JSON

1. Leer archivos

Usa with open("datos.txt", "r") as archivo: para leer txt.

2. Formato JSON

Ideal para guardar diccionarios en disco duro y cargarlos luego.

MAESTRO 3: P.O.O. Inicial

1. Clases y Objetos

Las Clases son los "moldes" y los Objetos son lo que creas con ese molde.

2. Método __init__

El constructor que inicializa los atributos de tu objeto cuando nace.

MAESTRO 2: Herencia

1. Clases Hijas

Crea nuevas clases que heredan todo de otra clase superior.

2. Función super()

Permite llamar métodos de la clase padre sin reescribir código.

MAESTRO 1: Métodos Mágicos

1. Dunder Methods

Métodos especiales como __str__ para definir cómo se imprime tu objeto.

2. Decoradores

Funciones que "envuelven" a otras para darles superpoderes extra.

CODE-PREDATOR: Nivel Máximo

1. Consumo de APIs

Usa la librería requests para traer datos de internet a tu app.

2. Bases de Datos (SQL)

Conecta Python con bases de datos como MySQL para proyectos reales.
